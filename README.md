# Seabrook Realty Pattaya

Official website for **Seabrook Realty Pattaya**, a real estate agency providing rental, sales, and client-focused property services in Pattaya, Thailand.

This repository contains the full source code for the public-facing website, including property listings, team information, and client inquiry forms.

---

## Purpose

The website is designed to:

- Showcase available rental and sale properties
- Present a clear and professional agency profile
- Collect and manage client inquiries efficiently
- Enable fast follow-up via email and messaging platforms
- Remain simple to update without a database or heavy CMS

The focus is on **clarity, speed, and ease of maintenance**.

---

## Features

- Property listing pages with individual detail views
- Client inquiry forms (name, phone, WhatsApp, LINE, Telegram, email)
- Team introduction and service overview
- Mobile-friendly, responsive design
- Fast global delivery via CDN
- Automatic SSL and secure deployment

---

## Tech Stack

- **Next.js** — React-based framework for performance and SEO
- **Tailwind CSS** — Utility-first styling for consistency and speed
- **Vercel** — Hosting, deployment, CDN, and SSL
- **Formspree** — Lead capture, storage, and email notifications
